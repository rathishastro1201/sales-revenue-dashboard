# Sales & Revenue Analysis Dashboard

## Project Overview

This project is an interactive Sales & Revenue Analysis Dashboard built in Microsoft Excel. It helps analyze sales performance, revenue trends, and top-performing products through charts, KPIs, and filters.

## Features

* Import data from Excel worksheets
* Automatic revenue calculations using formulas
* KPI tracking:

  * Total Sales
  * Total Revenue
  * Average Order Value
  * Growth Rate
* Revenue Trend Analysis
* Top-Selling Products Analysis
* Regional Sales Analysis
* Interactive Charts and Visualizations
* Filters and Slicers for dynamic reporting

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas

## Dashboard Components

* KPI Cards
* Revenue Trend Line Chart
* Top Products Bar Chart
* Sales by Region Pie Chart
* Interactive Filters
